# 内容生产机器框架

## 定位

一套 AI 驱动的全链路内容生产系统。输入一个选题，输出发布到全平台的成品视频。

## 全链路

```
选题 → 文案 → 声音 → 视频 → 分发
```

| 环节 | 方案 | 状态 |
|------|------|------|
| 选题 | AI Native 群信号提取 + 觅游热点 + GitHub Trending | 🔄 持续迭代 |
| 文案 | （待构建） | ⏳ |
| 声音 | 豆包 Seed Audio 1.0 / 小米声音模型 | 📡 已发现 |
| 视频 | OpenMontage（Prompt 驱动）+ Remotion/HyperFrames（Code 驱动） | 📡 已发现 |
| 分发 | social-auto-upload（全平台一键分发） | 📡 已发现 |

## 核心原则

1. **先用两周，再搭体系**——第一版粗糙但能用就行
2. **80% 力原则**——不是每条内容都要全力投入
3. **可逆/不可逆决策**——发内容可逆（发完能删），先发再说

## 赚钱路径

- 自己做内容 → 获取流量 → 变现
- 把生产工具卖给同样想做的人 → 直接变现
- 生产和卖工具的过程本身是内容 → 复利
